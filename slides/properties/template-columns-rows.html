Container Properties

grid-template-columns
grid-template-rows

	
.container {
	display: grid;
	grid-template-columns: 200px 200px 200px 200px;
	grid-template-rows: 175px 175px;
	grid-gap: 15px;
}
  • `grid-template-columns` and `grid-template-rows` define the line names and track sizing functions of the grid columns and rows, respectively.
  • In short, these rules are used to explicitly define your rows and colums.
