Container Properties

grid-auto-columns
grid-auto-rows

	
.container {
	display: grid;
	grid-auto-columns: 200px;
	grid-auto-rows: 175px;
	grid-gap: 15px;
}
  • `grid-auto-columns` and `grid-auto-rows` specify the size of an implicitly-created grid column and row, respectively.
  • So here, we're setting all implicitly-created items to be 200px by 175px.
