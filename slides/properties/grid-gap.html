Container Properties

grid-column-gap
grid-row-gap

	
.container {
	display: grid;
	grid-template-columns: 200px 200px 200px 200px;
	grid-column-gap: 10px;
	grid-row-gap: 15px;
}

Shorthand: grid-gap

	
.container {
	display: grid;
	grid-template-columns: 200px 200px 200px 200px;
	grid-gap: 10px;
}
  • `grid-row-gap` and `grid-column-gap` specifies the size of the grid lines, setting the width of the gutters between the columns/rows.
  • `grid-gap` is the shorthand for `grid-row-gap` and `grid-column-gap`.
