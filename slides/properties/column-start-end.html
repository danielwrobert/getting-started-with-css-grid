Item Properties

grid-column-start & grid-column-end

	
.item1 {
	grid-column-start: 1;
	grid-column-end: 3;
}

Shorthand: grid-column

	
.item1 {
	grid-column: 1 / 3;
}
  • The `grid-column-start` and `grid-column-end` properties specify a grid itemâ€™s start and end position, respectively, within the grid column.
  • Here, we're setting the element with the class of `item1` to span over two columns in width.
  • And `grid-column` is the shorthand property for the two, where the first value is the start position and the second property is the end position.
