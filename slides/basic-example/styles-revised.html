CSS

	
.container {
	display: grid;
	grid-template-columns: 200px 200px 200px 200px;
	grid-template-rows: 200px 200px;
	grid-gap: 20px;
}

Result

  • Here, we've set up 4 columns of 200px wide and 2 rows of 200px in height.
  • We've set the space between our rows and columns to be 20px, with the `grid-gap` rule.
  • The result is much better. And not too bad for just 4 lines of CSS!
  • But thre's still room for improvement. For example, how do we get the items to take up the full space? Calculate percentages?
