Container Properties

grid-auto-columns & grid-auto-rows

	
    .container {
        display: grid;
        grid-auto-columns: 200px;
        grid-auto-rows: 200px;
        grid-gap: 20px;
    }
  • The `grid-auto-columns` property specifies the size of an implicitly-created grid column.
  • While the `grid-auto-rows` property specifies the size of an implicitly-created grid row.
  • So here, we're setting all implicitly-created items to be 200px by 200px.
