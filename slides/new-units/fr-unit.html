The FR Unit

	
.container {
	display: grid;
	grid-template-columns: 1fr 1fr 1fr 1fr;
	grid-gap: 20px;
}
  • Fr is a new unit of length, which is short for “fraction” or "fractal unit".
  • MDN defines it as a unit which represents a fraction of the available space in the grid container.
  • In this example, we've re-written the `grid-template-colums` to use the fr unit instead of absolute pixel units (which aren't very flexible in this case).
