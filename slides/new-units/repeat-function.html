The repeat() Function

	
.container {
	display: grid;
	grid-template-columns: repeat(4, 1fr);
	grid-gap: 15px;
}

Result

  • We can improve on this further with the repeat() notation.
  • Instead of repeating the same value over and over, we are now wrapping that value in repeat().
  • The second argument of repeat() is the value that you want repeated.
  • The first argument of repeat() is the number of times you want to repeat that value.
  • This allows a recurring pattern to be written in a more compact form.
  • The result is the same as the longhand version.
  • Now that we've replaced the pixel values with a more flexible `fr` unit, we see that the grid items now take up the full width of the container.
  • What's happeining here is the `fr` units are telling the grid to fill in all of the available space with 4 evenly spaced columns.
